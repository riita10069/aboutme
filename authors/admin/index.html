roche

made by golang.
This is a powerful kit for creating microservices.
With a single command, the service is standardized.
It generates CRUD and migration code by only writing a .proto file.
It can also be an SDK for implementing middleware.
マイクロサービスの開発を助けてくれるGoのコードジェネレータ、Web Frameworkです。
.protoファイルを書くだけで、クリーンアーキテクチャに従ったCRUD処理をするコードとマイグレーションファイルを自動生成します。
また、APIの雛形を生成したり、ミドルウェアを統一するSDKとしても機能し、規律あるマイクロサービスの開発を助けます。

cynthia

made by golang.
Cynthia ensures test-driven development.
When you use it in CI, it ensures that untested methods are not introduced.
The reason for this is that Cynthia alerts you to a function that has no tests.

check_interface

made by golang.
check_interface warns against interfaces that are not implemented.
Interfaces that are not implemented in any method are the cause of the nill pointer exception.
静的解析を用いて、どのメソッドにも実装されていないインターフェイスに警告を出します。
それによって起こる実行時エラーを防ぎます。
